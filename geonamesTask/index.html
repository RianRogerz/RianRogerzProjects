Airport Weather Data Using an input of a specific airport ICAO code the API returns the weather station and the most recent weather observation for the inputted airport.
City Wikipedia Search Using an input of a specific popular UK city, the API returns the most relevant Wiki information regarding the city, with a hyperlink to the page, image from the page and a pinpoint on google maps using its coordinates.
Postcode Data Search By typing an input of a UK postcode with no spaces, the API will return the information where the postcode is located and a pinpoint on google maps using its coordinates.
